# Screenshot Requirements for Store Submission

## Apple App Store (minimum set)

Prepare screenshots for at least one iPhone size. Recommended sets:

- 6.7-inch display (required/recommended primary)
- 6.5-inch display (optional fallback)
- iPad 12.9-inch (if iPad support is claimed)

Recommended screenshot count per locale:

- 5 to 8 screenshots

Suggested screenshot sequence:

1. Unified support inbox
2. Ticket list and status overview
3. Ticket detail and assignment flow
4. Customer profile/history
5. AI-assisted support workflow
6. Omnichannel integration screen

## Google Play Store (phone)

Required:

- Phone screenshots: 2 to 8
- Minimum dimension: 320 px
- Maximum dimension: 3840 px
- Aspect ratio: between 1:2 and 2:1

Recommended screenshot count:

- 6 to 8 screenshots

Suggested sequence aligns with App Store set for consistency.

## Quality Guidelines

- Use production-like data (no lorem ipsum)
- No debug banners or development artifacts
- Keep language consistent with store locale
- Avoid showing sensitive customer information
- Keep captions short and benefit-oriented
