# Mobile Store Publishing Toolkit

This folder contains checklists and templates for Issue #17:

- Publishing the app to Apple App Store and Google Play Store.

## Files

- issue-17-execution-checklist.md: End-to-end checklist
- store-listing-copy.en.md: Draft listing copy
- screenshot-requirements.md: Screenshot matrix and quality guidance
- android-release-signing.md: Android signing setup and AAB build
- ios-release-signing.md: iOS signing and App Store upload flow
- submission-tracker-template.md: Review tracking template
- commands.md: Common release commands

## Recommended workflow

1. Prepare assets and listing copy
2. Configure signing
3. Build and upload artifacts
4. Submit to stores
5. Track review feedback until both apps are live
